🔧 FIX 1: HTTP PROXY CONNECTION ERROR

Problem: HTTP proxy connect ho rahi thi but websites open nahi ho rahe the
Cause: DNS resolution proxy ke through nahi ja raha tha (direct system DNS use ho raha tha)

Solution Applied:
- Added: --host-resolver-rules="MAP * ~NOTFOUND , EXCLUDE localhost"
- This forces ALL DNS queries through proxy
- Now websites will load properly through HTTP proxy ✅
